# Homelab-Pathways
*Want to break into IT? Don't know where to start?* With this quick guide, you can get started with your own home lab with **simple, easy to use hardware and software** that will help you get started on your journey. If you're and experienced IT pro and want some nice everyday locally hosted services, feel free to use this guide as well.

For an example, I started my IT education in 2021 shortly before getting a job in data analytics and coding using SAS and SQL and simulataneously studying for the Comptia Security+ certification. I started building out homelab components to enhance my knowledge in the field and gain an understanding of how role based permissions work, how networking concepts work in practice, and discover what is needed for all of the services we use everyday.
